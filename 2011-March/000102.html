Mike Gabriel <mike.gabriel at das-netzwerkteam.de> (2011-03-01 21:11) wrote:

Have you installed PyHoca-GUI with the latest NSIS-based installer package?
http://code.x2go.org/gitweb?p=pyhoca-contrib.git;a=tree;f=win32-dist/installers;h=0f7b34077de60296bf130f958787c4c92805e045;hb=c034a0cd030534d169d505b4b3bc936ee251d35b

If yes: please make sure that either Xming 6.9.x, Cygwin/X or the very  
latest VcXsrv is also installed. It is also recommended to install  
Ghostscript/Ghostgum on the client (for PDF handling of incoming  
printjobs).

If you start pyhoca-gui from the command line, please make sure you  
use at least the following options:

   pyhoca-gui.exe --start-xserver --start-pulseaudio --quiet

If you follow these instructions, do you still observe the same issue?
